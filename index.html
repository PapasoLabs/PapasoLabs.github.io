Decentralized Banking Protocol

Nowadays two individuals needs a third party to create a financial agreement. The nature of this behavior is based on trust, in particular the lack of trust between each peer in the transaction.

The iron bank network can solve this lack of trust using smart contracts, which automates the agreement and allow peer-to-peer interactions between users in the blockchain. This new paradigm allows more self-manage financial relationships, making decentralized banking a reality.

How it works

The iron Bank Network is a blockchain infrastructure running by public smart contracts that can be accessed by any peer in the network. These contracts allow users to make agreements with any other counterpart without any third party involved.

Therefore, this project aims to change how Banking works through a Trustless system. The blockchain uses the Ethash Dagger-Hashimoto algorithm to generate a token called IRON. The computational power behind this token supports all the operations needed in our infrastructure (e.g. create an account, send/receive IRON, create a Loan). Every transaction service made by a Bank today can be implemented within this blockchain for Free.

All the infrastructure is based on Ethereum Smart Contracts written in Solidity. This ecosystem enables users to exchange peer-to-peer services in a frictionless environment whereas immutability and trust is secured.
